2. **需要安装 Rust** - 访问 https://www.rust-lang.org/
3. **网络问题** - 建议使用淘宝镜像
4. **Windows 特定** - 需要 WebView2 和 VS Build Tools

### 💡 开发建议

1. **推荐的开发流程**:
   - 先实现截图功能
   - 再集成 Tesseract

   - F12 打开浏览器开发工具
   - Rust 错误看终端

3. **性能优化**:
   - 首次编译后会快很多
